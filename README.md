# Telegram Bot

Telegram bot parser images from different photo-stocks. It's excelent gui for designers. Also Bot includes inner registration

---

## Tech Stack

**Main**

- Python
- aiogram

**Server**

- MongoDB
- Redis

**misc**

- httpx
- googletrans
- fake_useragent
